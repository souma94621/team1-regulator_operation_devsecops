# Для проверки сторонней системы регулятором
У вас в корне репозитория должен лежать файл regulator.yml
### regulator.yml
```
version: "1.0"

coverage_command: "pytest tests/unit/"

security_goals:
  - "FW-SEC-01"
  - "FW-SEC-02"
  - "FW-SEC-03"
  - "SYS-SEC-01"

components:
  trusted:
    - "src/policy_checker.py"
    - "src/claim_validator.py"
  untrusted:
    - "src/api.py"
    - "src/kafka_adapter.py"
```
