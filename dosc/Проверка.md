# Для проверки сторонней системы регулятором
У вас в корне репозитория должен лежать файл regulator.yml
#### regulator.yml
```
version: "1.0"

coverage_command: "pytest tests/unit/" -- путь к юнит тестам

security_goals: -- соблюдаемые целм безопасности, названия в соответствии с реестром
  - "FW-SEC-01"
  - "FW-SEC-02"
  - "FW-SEC-03"
  - "SYS-SEC-01"

components: -- список компонентов
  trusted: -- доверенные
    - "src/component_a/src/component_a.py"
    - "src/component_b/src/component_b.py"
  untrusted: -- недоверенные
    - "src/component_c/src/component_c.py"
    - "src/component_d/src/component_d.py"
```
